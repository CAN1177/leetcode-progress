# 双指针 / Double Pointers

## 一句话定义
使用两个变量（指针）在序列（数组、字符串、链表）上按特定规则移动，通过缩小搜索空间将 $O(n^2)$ 降低到 $O(n)$。

## 核心不变量
- **对撞指针**：左右指针指向的元素满足对称性或目标和的关系。
- **快慢指针**：慢指针左侧（或快慢之间）维护一个满足特定性质的区间。

## 触发信号（气味清单）
题目里出现以下任一线索，就应该立刻想到这个模式：
- 线索 1：判断字符串或数组的对称性（回文）。
- 线索 2：有序数组中找两个数之和。
- 线索 3：原地修改数组（删除重复项、移动元素）。
- 线索 4：链表找中点或判断是否有环。

## 反触发信号（看起来像但其实不是）
- 例：寻找所有可能的子序列组合 → 通常是回溯或 DP，双指针处理不了非连续或非线性缩小的空间。

## 最小骨架（对撞指针版）
```javascript
let left = 0;
let right = s.length - 1;
while (left < right) {
    // 1. 跳过无效元素 (如果是字符串处理)
    while (left < right && !isValid(s[left])) left++;
    while (left < right && !isValid(s[right])) right--;

    // 2. 核心判断
    if (s[left] !== s[right]) return false;

    // 3. 收缩边界
    left++;
    right--;
}
return true;
```

## 最小骨架（快慢指针版）
```javascript
let slow = 0;
for (let fast = 1; fast < nums.length; fast++) {
    // 1. 寻找符合条件的元素
    if (nums[fast] !== nums[slow]) {
        // 2. 扩展有效边界并覆盖
        slow++;
        nums[slow] = nums[fast];
    }
}
return slow + 1;
```

## 常见变体
| 变体 | 关键差异 | 代表题 |
| :--- | :--- | :--- |
| 对撞指针 | 左右向中间靠拢 | 验证回文串、两数之和 II |
| 快慢指针 | 同向但速度/起点不同 | 移动零、删除有序数组重复项 |
| 滑动窗口 | 快慢指针维护一个窗口 | 最小覆盖子串 |

## 踩过的坑
- 内层跳过逻辑忘记检查 `left < right` 导致指针擦肩而过。
- 快慢指针中 `slow` 的初始位置和 `return` 的长度关系。
- 多变量求和中的去重位置（尤其是外层循环 `i` 的去重）。
