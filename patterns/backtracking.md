# 模式：回溯算法 (Backtracking)

## 核心心法
回溯是 DFS 的一种特例，本质是**在决策树上进行深度优先遍历**。它的核心在于“状态重置”，即在尝试完一种可能性后，将环境恢复原状，以便尝试下一种可能性。

## 核心不变量
- **路径 (Path)**：已经做出的选择。
- **选择列表 (Options)**：当前还可以做的选择。
- **结束条件 (End Condition)**：到达决策树底层。

## 最小骨架
```javascript
const result = [];
function backtrack(路径, 选择列表) {
    if (满足结束条件) {
        result.push([...路径]);
        return;
    }
    
    for (let 选择 of 选择列表) {
        做选择;
        backtrack(路径, 选择列表);
        撤销选择;
    }
}
```

## 经典题型对照
1. **组合/子集问题**：每个节点都是解，通过 `start` 索引去重。
2. **排列问题**：叶子节点是解，通过 `used` 数组去重。
3. **切割问题**：本质是组合问题，切割点即 `start`。

## 避坑指南
- **拷贝问题**：JS 中记档必须 `[...path]`。
- **剪枝 (Pruning)**：对于有重复元素的情况，必须先排序，再通过 `if (i > start && nums[i] == nums[i-1]) continue` 跳过。
