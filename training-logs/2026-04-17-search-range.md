## 核心不变量
"寻找第一个位置时，一旦 `nums[mid] === target`，立刻收缩右边界 `right = mid - 1` 以寻找更左侧的可能；寻找最后一个位置时，则收缩左边界 `left = mid + 1`。"

## 攻击向量
通过两个几乎相同的二分查找过程（或者一个带参数的通用函数），分别定位左边界和右边界。利用 `answer` 变量记录当前找到的最优下标，保证逻辑稳健。

## 模式 + 为什么这个模式适用
二分查找。由于数组有序且要求 $O(\log n)$ 复杂度，二分是唯一选择。与基础二分不同的是，找到目标后不停止，而是利用“单调性”继续收缩搜索空间。

## 最小骨架（寻找边界版）
```javascript
function findBound(nums, target, isLeft) {
    let left = 0, right = nums.length - 1, ans = -1;
    while (left <= right) {
        let mid = left + Math.floor((right - left) / 2);
        if (nums[mid] === target) {
            ans = mid;
            if (isLeft) right = mid - 1; // 寻找左边界，向左逼近
            else left = mid + 1;        // 寻找右边界，向右逼近
        } else if (nums[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return ans;
}
```
