# 33. 搜索旋转排序数组 (Search in Rotated Sorted Array)

- **日期**: 2026-04-22
- **模式**: [Binary Search](../patterns/binary-search.md)
- **难度**: 中等
- **状态**: 复习完成 (核心逻辑已攻克)

## 训练记录

### 1. 核心不变量
在一个旋转排序数组中，取任意中点 `mid`，左半部分 `[left, mid]` 和右半部分 `[mid, right]` 中，**必然至少有一个是严格升序的**。

### 2. 触发信号
- 有序数组（或局部有序数组）。
- 需要在 $O(\log n)$ 时间复杂度内完成查找。

### 3. 关键感悟
- **局部有序的利用**：二分查找不一定要求全局单调。只要能根据 `mid` 划定一个“确定性有序”的区间，就能利用该区间来排除掉另一半搜索空间。
- **安全的中点计算**：习惯使用 `left + (right - left) / 2` 替代 `(left + right) / 2`，展现对底层整数溢出风险的防范意识。
- **排除法的精髓**：二分查找的本质不是“找”，而是“删”。每一轮删掉一半不可能的区间，剩下的就是真相。
