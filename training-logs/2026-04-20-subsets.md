## 核心不变量
"在回溯递归树中，**每一个节点**（节点即状态）都是一个合法的子集。通过 `start` 索引控制选择列表的范围，确保元素按顺序选取，从而天然避开重复组合。"

## 攻击向量
使用 DFS 遍历决策树。在递归的每一层，决定是否将当前元素加入 `path`。由于本题要求所有子集，因此在 `backtrack` 函数入口处即刻记档当前 `path`。

## 模式 + 为什么这个模式适用
回溯算法。适用于“搜索所有可能解”的问题。通过“做选择 -> 递归 -> 撤销选择”的循环逻辑，高效遍历 $2^n$ 的解空间。

## 最小骨架（回溯子集版）
```javascript
function backtrack(start, path) {
    res.push([...path]); // 每一个节点都是解
    for (let i = start; i < nums.length; i++) {
        path.push(nums[i]);  // 做选择
        backtrack(i + 1, path); // 递归
        path.pop();          // 撤销选择
    }
}
```

## 触发信号
- "返回所有可能的子集/幂集"
- "结果中不能包含重复的子集"
- 元素互不相同（基础版回溯）

## 关键失误
- **引用传递陷阱**：直接 `res.push(path)` 会导致结果全为空，必须使用 `[...path]` 进行拷贝。
- **递归参数错误**：误将 `i + 1` 写成 `start + 1`，会导致元素重复使用。
